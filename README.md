# Mastering Big Data Analytics with PySpark 🚀

Este repositorio contiene los proyectos y ejercicios realizados durante el curso **"Mastering Big Data Analytics with PySpark"** impartido por Danny Meijer (Packt Publishing). El objetivo principal es escalar el análisis de datos y construir pipelines de Machine Learning robustos utilizando el ecosistema de Apache Spark.

## 📊 Sobre el Curso
El curso se enfoca en aprovechar el potencial de **PySpark** para procesar conjuntos de datos masivos que superan las capacidades de las herramientas tradicionales (como Pandas). Abarca desde la arquitectura de Spark hasta el despliegue de modelos escalables.

### Temas Clave:
* **Arquitectura de Spark:** Entendimiento de Driver, Workers y procesamiento distribuido.
* **Spark SQL & DataFrames:** Consulta y manipulación de Big Data de forma eficiente.
* **Spark MLlib:** Creación de pipelines de Machine Learning (como sistemas de recomendación ALS).
* **Spark Streaming:** Procesamiento de datos en tiempo real con alta tolerancia a fallos.
* **Visualización:** Integración de Jupyter con Spark para análisis exploratorio (EDA) a escala.
